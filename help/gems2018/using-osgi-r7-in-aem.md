# 在AEM中使用OSGi R7{#using-osgi-r-in-aem}

OSGi R7版本体现了许多重要的新特性和功能，并为许多对Java开发人员重要且易于使用的现代化技术提供了基于标准的开放方法。  新增功能包括（及其他）：

* 声明性服务更新
* 新包注释
* 推送流和承诺
* 转化因素
* Http Whiteboard
* 配置器和配置管理员
* Java 9支持

在本次研讨会中，我们将重点介绍并解释OSGi R7中最有趣的新增功能，重点介绍可以与AEM 6.4一起使用的功能。
