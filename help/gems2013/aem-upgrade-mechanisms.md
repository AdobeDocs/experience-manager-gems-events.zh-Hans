# AEM 5.6升级机制 {#aem-upgrade-mechanisms}

此会话介绍了CQ/AEM中的升级机制，这些机制允许仅通过将CQ/AEM jar文件替换为新的文件来升级CQ本身以及自定义应用程序和内容。 我们将介绍一般升级方案，描述升级扩展点，解释我们如何测试升级机制，并评论与升级相关的一些技术挑战。
