# AEM 6.0开发人员模式{#aem-developer-mode}

David将介绍AEM 6.O中的开发人员模式。开发人员模式是：

* 作者与开发人员之间的桥梁（组件树）
* 调试助手（错误通知程序）
* 测试伴侣（应用程序测试工具）(Hobbes.js AEM实施)
