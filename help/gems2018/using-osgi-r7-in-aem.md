# 在AEM中使用OSGi R7{#using-osgi-r-in-aem}

OSGi R7版本代表了许多重要的新特性和功能，并为许多对Java开发人员而言重要的现代有价值且简单易用的技术提供了基于标准的开放方法。  新增功能包括（其中包括）：

* 声明性服务更新
* 新建捆绑批注
* 推送流和承诺
* 转换器
* Http白板
* 配置器和配置管理员
* Java 9支持

在本课程中，我们将重点介绍OSGi R7中最有趣的新增内容，重点介绍可以与AEM 6.4一起使用的功能。
